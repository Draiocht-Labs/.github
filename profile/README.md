## ▌ Overview

Draíocht Labs is a FiveM development team building resources, frameworks, and tooling for roleplay servers. We work across the QBCore and Qbox ecosystems, focusing on clean, maintainable Lua/TypeScript codebases and performance-conscious scripting.

```
team:        Draíocht Labs
focus:       FiveM server-side & client-side development
frameworks:  QBCore, Qbox
languages:   Lua, TypeScript, JavaScript
status:      active development
```

## ▌ Tech Stack
